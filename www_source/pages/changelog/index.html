Changelog

0.5.0

  • Updated to use Knockout 3.2
  • Removed dependency on when.js in favor of using Promise polyfill
  • Added ability to nest viewmodels inside each other

This version removes the dependency on when.js in favor of using a Promise polyfill. This reduces the complexity of the require config file and promotes the use of an ES6 standard above the use of libraries. If you are upgrading from 0.4.0 you need to remove the reference to the when.js package from the require config file and if you are using a browser without ES6 promise support you will have to add the polyfill to the page as a script tag.
